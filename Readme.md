## Short description

### Install
```
mv ~/.vimrc ~/.vimrc.old
ln -s $(path_repo)/.vimrc ~/.vimrc
```

### Shotcuts

 - Ctrp+p: (CtrlP) Search file in all subdirs
 - Ctrl+\\ + d: (Cscope) Search definition of current word
 - Ctrl+o: Open a terminal
 - Ctrl+f: Focus current file in filetree
